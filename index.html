📊 Dognition Engagement Analysis

This analysis explores how behavioral and demographic factors influence user engagement in a freemium model.

🐶 Personality Dimension vs Test Completion

Dogs across different personality dimensions completed a similar number of tests (~23 tests per dog). This suggests personality traits do not strongly influence engagement behavior.

💡 Key Insight: Engagement is consistent across personality types, indicating that product experience—not inherent traits—drives user activity.

🐕 Breed Type vs Test Completion

Breed type shows only slight variation in engagement levels. Popular Hybrid dogs completed marginally more tests, while Mixed Breeds completed slightly fewer.

💡 Key Insight: Breed type has minimal impact on engagement, suggesting broad product-market fit across user segments.
